# Проект 1: Музыка больших городов
## Данные
- Идентификатор пользователя
- Название трека
- Исполнитель
- Жанр
- Город
- Время прослушивания
- День прослушивания.

Провести сравнение музыкальных предпочтений у пользователей Яндекс.Музыки, проживающих в Москве и Санкт-Петербурге.
